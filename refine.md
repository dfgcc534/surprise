# 가윤이 생일 게임 — 시퀀스 & 에셋 논의 기록

## 전체 방향

- UI/UX: 샌즈(Undertale) 스타일
  - 바텀 박스 대화창 (화면 하단 고정)
  - 캐릭터는 화면 상단/중앙 배치
  - 느릿한 타이핑 대사
  - 장면 전환 (페이드/슬라이드)
  - 전투 메뉴 등 게임 진행 방식은 따라하지 않음
- 분위기: 악당 등장 직전까지 로맨틱, 악당 등장 이후 게임 풍, 클리어 후 다시 로맨틱
- 컨셉: 고양이 배달원이 편지를 전달하러 온 느낌

## Step별 시퀀스

### Step 1 — 인트로
- 캐릭터: stand_cat.png (처음부터 뜬 상태)
- 대사 (타이핑 효과, 쭈뼛쭈뼛 배달원 톤):
  1. "저기..."
  2. "배달... 왔는데요..."
  3. "혹시... 이가윤 씨 맞으세요?"
  4. "편지를 받으시겠습니까?"
- 선택지: [Y] [N]
- N 클릭 시: "...받아주세요" 텍스트 교체 후 버튼 유지
- Y 클릭 시: goToStep(2)

### Step 2 — 본인 확인
- 캐릭터 전환: stop_cat.png → IMG_3053.PNG (안경)
- 대사 시퀀스:
  - "잠깐요!" — stop_cat.png
  - "수령 확인이 필요합니다" — stop_cat.png
  - "몇 가지 질문에 답해주시겠어요?" — IMG_3053.PNG로 전환
- 버튼 (마지막 타이핑 완료 후 등장):
  - [퀴즈를 푼다] → goToStep(3)
  - [거절한다] → 게임오버 시퀀스 → goToStep(1)

### Step 3 — 퀴즈
- 캐릭터: IMG_3053.PNG (안경) 유지
- 형식: Y/N 객관식 버튼
- 질문 3개:
  1. "안세호는 이가윤을 사랑합니까?" → yes
  2. "이가윤은 안세호를 사랑합니까?" → yes
  3. "14cm 이상?" → yes
- 정답 → 다음 문제, 전부 통과 시 goToStep(4)
- 오답 → Step 3-F

#### Step 3-F — 퀴즈 실패
- 캐릭터: gun_cat.png
- 기존 대사 유지: "어딜 속이려고!"
- GAME OVER 연출 → goToStep(1)

### Step 4 — 편지 전달 (로맨틱 클라이맥스)
- 캐릭터: stand_cat.png → 편지 건넬 때 Untitled.png (양손)으로 전환
- 대사:
  1. "확인되었습니다..."
  2. "오래 기다리셨죠..."
  3. "소중히 전달해 드리라고 했어요"
- 대사 완료 후 캐릭터 Untitled.png로 전환 + 고양이가 편지를 들고 있는 연출
- 추가 대사: "여기요..." (타이핑 효과)
- 이후 편지가 화면 중앙에 크게 등장 (180px)
- 봉투 위에 "CLICK" 글씨 표시
- 사용자가 봉투 클릭 → goToStep(5) (악당 등장)

### Step 5 — 악당 등장 + 편지 탈취 (분위기 반전)
- 연출: 화면 흔들림 + 빨간 플래시
- 악당(강아지) 오른쪽에서 슬라이드 인: assets/bad/제목 없음 (6).png
- 악당 대사: "가윤이는 내 꺼야!!!"
- 편지가 악당 쪽으로 날아가는 애니메이션
- 고양이 → IMG_3052.PNG (우는 표정)으로 전환
- 고양이 대사: "으아아..."
- 타이머로 자동 → goToStep(6)

### Step 6 — 가윤이 캐릭터 등장
- 화면: 고양이(IMG_3052.PNG, 우는 표정) 남아있는 상태
- 가윤이 등장 (왼쪽에서 슬라이드 인): assets/you/image.png (기본)
- 대사:
  1. "울지 마..." — image.png (기본)
  2. "내가 꼭 찾아줄게!" — image copy 2.png (전투 포즈)로 전환
- 버튼: [출발!] → goToStep(7)

### Step 7 — 런 게임
- 사이드스크롤 러닝 게임
- 가윤이가 악당(강아지)을 쫓아감

#### 조작
- 오른쪽 방향키: 점프
- 왼쪽 방향키: 슬라이딩
- 화면 오른쪽 아래에 조작법 안내 텍스트 작게 표시

#### 게임 양식
- 구글 오프라인 런 게임(크롬 공룡) 참고
- 가윤이 화면 왼쪽 고정, 배경/장애물이 오른쪽에서 왼쪽으로 스크롤
- 클리어 거리 도달 시: 장애물 멈추고, 가윤이가 화면 내에서 오른쪽으로 이동 → 화면 오른쪽 끝에 악당이 있고 따라잡는 연출

#### 가윤이 스프라이트
- 달리기: assets/you/jumping/image copy 3.png (옆모습)
- 점프: assets/you/jumping/ (여러 프레임 애니메이션)
- 슬라이딩: assets/you/sliding/ (여러 프레임 애니메이션)
- 피격: assets/you/die/ (사망 포즈)

#### 피격
- 무적 시간 1초 (깜빡임 연출)

#### 장애물
- 지상 (점프로 회피): assets/obstacle/ground/ (좀비, 나무 그루터기 등)
- 공중 (슬라이딩으로 회피): assets/obstacle/sky/ (UFO 등)

#### 음식 트랩
- assets/obstacle/ground/food/ 에셋 사용
- 화면에 작게 표시, 캐릭터가 닿으면:
  - 나레이션: "공주는 먹는 것에 정신이 팔려 그만 편지에 대해 잊어버리고 말았다..."
  - 목숨 무관, 바로 Step 7 재시작

#### 크기 설정 (데스크톱 기준)
- 플레이어: 120x165 (서있기), 165x80 (슬라이딩)
- 지상 장애물: 100x100
- 공중 장애물: 95x95
- 음식 트랩: 65x65
- 악당: 140x140
- 점프력: -17, 중력: 0.8 (최대 높이 ~180px)

#### 난이도 & 진행
- 목숨 3개 (화면에 하트로 표시)
- 시간 지남에 따라 속도 증가
- 화면에 진행도 바 표시
- 일정 거리 달리면 클리어

#### 게임오버 / 클리어
- 목숨 소진 시: Step 7 처음부터 재시작
- 클리어 → 악당 잡는 연출 → goToStep(8)

### Step 8 — 편지 탈환 + 축하
- 가윤이: image copy.png (양손 올림) — 편지 들어올리는 애니메이션
- 고양이 다시 등장: IMG_3054.PNG (미소)
- 고양이 대사: "역시 QUEEN 이야~"
- confetti 연출 (하트/별 낙하)
- 버튼 → goToStep(9)

### Step 9 — 편지 오픈 엔딩
- 캐릭터: Untitled (1).png (꽃 든 고양이) 옆에 배치
- 편지: 제목 없음 (17).png (봉인된 봉투) 중앙
- 클릭 → 제목 없음 (14).png (열린 봉투)로 전환
- 제목 없음 (15).png (편지지) 펼쳐지며 편지 내용 타이핑
- 편지 내용: assets/letter.txt (나중에 채움)
- 마지막: "생일 축하해" 마무리 연출
